    I worked in collaboration with Harvard researchers to develop an R package for video annotation.     Automated Web Srapper

    This application, which is based on the Web Scraping technology, makes available in Excel format the climatological data for the city and the period of choice, as well as the corresponding DJU calculations. The data that comes from the French climatology website (infoclimat.fr) via Web Scraping are updated every minute with a Linux Cron JOB making the extration data updated in real time. This tool helps my client (an energy company) saving their time and avoid manual data entry.
